## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Local API Setup 

1. install mongodb community edition via brew install
2. install mongodb compass as a db UI
3. start mongodb `brew services start mongodb-community@5.0`
4. create a local connection in mongodb compass
5. in the terminal of mongodb compass type `use admin`
6. then type `db.createUser( { user: "root", pwd: "root", roles["readWrite"] })`
7. install docker
8. run `docker-compose -f docker-compose.dev.yml up`
9. run `npx ts-node src/seeds`
10. check on mongodb compass if data is in db instance 
11. add profile `seatti` to AWS credentials file using the access key and secret access key of your AWS account
